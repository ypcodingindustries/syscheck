# Security Policy

## Supported Versions
We currently only support the latest version of Syscheck running on **Fedora 43 Cloud**.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability
**Do not open a public GitHub issue for security vulnerabilities.**

If you discover a security hole in the dashboard or the engine (e.g., a way to bypass the Fail2Ban tracking or an RDP/SSH leak), please report it privately.

Please contact the maintainer via the email associated with the **ypcodingindustries** GitHub profile. We aim to respond within 48 hours to coordinate a fix.
